# Security Policy

## Supported Versions

Only the latest released version is supported with security updates.

## Reporting a Vulnerability

Please do **not** open a public issue for security reports.

Instead, use **GitHub Security Advisories** for this repository:

1. Go to the repository on GitHub.
2. Open the **Security** tab.
3. Click **Report a vulnerability**.

When reporting, please include:

- A clear description of the issue and impact
- Steps to reproduce (proof-of-concept if possible)
- Affected versions / commits (if known)
- Any relevant logs (please redact secrets / tokens)

We will respond as soon as practical and coordinate a fix and disclosure timeline.
