# Rollout Plan

## Phase 1: Shell + Device Context
- App shell, navigation, device selector, onboarding states.

## Phase 2: Core Tasks
- Dashboard quick actions, logcat viewer, file manager (read-only), wireless pairing.

## Phase 3: Power Features
- UI Inspector, live mirror mode, apps manager, advanced actions, command palette.

## Phase 4: Polish & Accessibility
- Full keyboard coverage, theme toggle, performance tuning.
